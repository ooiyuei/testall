## 設計トークン（color / type）の追加方針

### Color
- **退場**: `peach-*` → 用途別に振り分け
  - 警告・知識不足・誤答 → `coral-*`
  - 情報バッジ → `sky-*`
  - 達成・完了 → `mint-*`
- **存続**: `sky` (primary) / `mint` (success/streak) / `coral` (warning) / `ink` (text) / `cream` (bg)

### Typography
- 日本語見出し: `tracking-[-0.025em]`（既に多くで適用済み）
- 英字ラベル: 小文字 + `text-[11px] tracking-[0.08em]`（uppercase 廃止）
- 本文: `tracking-[-0.02em]`（PDF原則 02 より）
