# Part 0: 客観評価

**正直に言う。良いところも、ダメなところも。**

## スコア

| 軸 | スコア | コメント |
|----|--------|----------|
| 自然さ（口語か） | **7/10** | 一部に書き言葉が残っている |
| 説得力（根拠が通るか） | **6.5/10** | 統計の出典不明 + トラクションが薄い |
| ストーリーの一本線 | **6.5/10** | 感情ブロックとビジネスブロックが断絶している |
| 記憶に残るか | **8/10** | キーフレーズは強い |
| 行動喚起（Actionに繋がるか） | **6/10** | ASKとVisionの順序が逆 |
| **総合** | **6.8/10** | 開幕と核心は強い。構造と細部に修正が必要 |

---

## 強いところ（絶対に変えるな）

**1. 開幕ナラティブは完璧**
```
「テストが返ってきた。点数を見て、引き出しにしまった。永遠に出さない。」
```
映像的、普遍的、テンポがある。5分版でも1分版でも核はここ。

**2. 核心インサイトは刺さる**
```
「間違えたところにこそ、答えがある。」
```
短い。断定。記憶に残る。これがTestallの全てを表している。

**3. 競合フレーム「右上の空白」は正確**
2x2マトリクスで「誰も手をつけていない象限」を視覚化する。投資家が好む論理。

**4. タグラインは強い**
「あなた専用の家庭教師を、手のひらに。」= 直感的に何をするサービスかわかる。

**5. クロージングは残る**
「次の25分だけ、勝て。」— 短い、強い、覚えられる。

## 問題点（直すべき5つ）

### 問題1: ASKとVisionの順序が逆
**現状**: ASK（Slide 24）→ Vision/Why Us（Slide 25）→ クロージング
**問題**: 感情のピーク（Vision）の前にお金の話をすると、気持ちが冷める。
**正解**: Vision/Why Us → クロージング暗背景（感情最大値） → ASK

---

### 問題2: "模試後3日以内に振り返るのは2割以下" — 出典がない
**現状**: 出典不明のまま使っている。
**問題**: 投資家に「根拠は？」と聞かれたら詰まる。ピッチでの信頼喪失は致命的。
**正解A**: 正確な出典を付ける（ベネッセ・旺文社等の調査を探す）
**正解B**: 「多くの高校生が振り返らないまま次に進む」という言い方に変えて逃げる
**正解C**: 自分のヒアリング結果「30人中、模試を翌日に見直した人は5人だった」に変える（一次情報は最強）

---

### 問題3: "Testallの設計は4原則で貫かれています" — 書き言葉
**現状**: 「Testallの設計は4つの原則で貫かれています。楽であること、極限まであなた専用であること…」
**問題**: 会社パンフレットの文体。誰もこういう話し方はしない。
**正解**: 「作るときに絶対に曲げなかったのが4つあって——楽であること、完全にあなた専用であること、ゲーム感覚であること、ゴールが見えること。」

---

### 問題4: ブルームの2シグマが唐突に登場する
**現状**: Slide 25 Why Us の中で突然「1984年、教育学者ブルームが証明しています」
**問題**: 流れが切れる。「急に何？」という印象を与える。ブルームを知らない審査員も多い。
**正解**: ブルームは5分版のビジョンスライドに自然な形で組み込む。「個別指導の効果は40年前から証明されていて、問題はずっとコストだった。それを解くのがTestallです。」という形で。または1分・3分では省いてよい。

---

### 問題5: 感情ブロックとビジネスブロックの断絶
**現状**: Slide 15（コアループ）→ Slide 16（ターゲット「58万人」）
**問題**: 「楽しくなる、そこまで持っていくのがTestallです」という感情の高まりが、
「ターゲットは偏差値55-60の続かない高2」という冷静な話に突然切り替わる。
**正解**: 橋渡しの1文を入れる。
「このループが回る高校生が、日本に**58万人**います。」
これがあれば感情→ビジネスの移行が自然になる。

---

## 直せば7/10 → 8.5/10 になる

上の5問題を直すだけで説得力が大幅に上がる。開幕と核心は既に一流水準。

| エレベーター・名刺交換後 | **1分版** |
| 書類選考通過後の最初の面談 / ピッチイベント | **3分版** |
| ICCカタパルト / B Dash / DemoDayメイン発表 | **5分版** |
| 質疑応答が長い場合 | **3分版 + Q&A濃縮** |

**練習の順序**:
1分版を完璧にしてから3分版、3分版を完璧にしてから5分版へ。
短い版を完璧にした人間が、長い版も完璧にできる。逆は成立しない。
