## 設計の核（参考スライド：dinii）

### 学ぶべき5つの原則

1. **余白の量で語る** — 画面の40%以上を空白にする。詰めない。
2. **装飾しない** — 色は最小限、装飾線は1スライド1本まで、絵文字なし
3. **文字組が主役** — タイポグラフィの強弱が情報設計そのもの
4. **数字は孤立させて巨大に** — 「20」「324」「5万人」を画面の30%サイズで
5. **セクション意識** — 全スライドに小さなセクションラベル（左上）と通し番号（右下）

### 避けるべき5つのアンチパターン

1. ❌ 装飾線が文字に被る
2. ❌ 画像が画面端で切れる
3. ❌ 同心円・グラフのラベルが重なる
4. ❌ 蛍光ペン強調が句点と被る
5. ❌ 1スライドに3つ以上のメッセージを詰める

「あなた専用の家庭教師を、手のひらに。」というキャッチを持つ EdTech スタートアップ Testall のシードラウンド向けピッチデッキを 23 スライド作成してください。デザインの参考は **dinii / Linear / Notion / Stripe** のような、余白の多い極めてミニマルな日本スタートアップの投資家向けスライド。

## デザインシステム（全スライド共通）

### カラーパレット（厳守）
- **背景（標準）**: `#F8F5EE`（温かみのあるオフホワイト・クリーム）
- **背景（感情ピーク）**: `#0B0E13`（深い黒、純黒は避ける）
- **テキスト主**: `#0A0A0A`
- **テキスト副**: `#6B6B6B`
- **テキスト極淡**: `#A8A8A8`
- **アクセント青**: `#1A6FFF`（タグライン強調、ハイライト）
- **アクセント橙**: `#FF6B35`（重要数字、トランジション）
- **これ以外の色は使わない**

### フォント
- 日本語: **Noto Sans JP**（Black / Bold / Medium / Regular の4階層）
- 英数字: **Inter**（Bold / Medium / Regular）
- **コントラスト**：見出しは Black 重量、本文は Regular。Medium は避ける。

### レイアウトグリッド
- アスペクト比: **16:9**（1920×1080px）
- 余白: 上下左右 **80px 以上**
- セクションラベル位置: 左上 80px 内側
- ページ番号位置: 右下 80px 内側
- ロゴ位置: 左下 80px 内側

### 全スライド共通要素
- 左上に小さなセクションラベル（11px / `#6B6B6B` / 大文字英字 + 日本語）
  - 例：`— SCENE 01 / 開幕`
- 右下に通し番号（11px / `#A8A8A8`）
  - 例：`02 / 23`
- 左下に小さな Testall ロゴマーク（青の四角に T、12px）+ "Testall" 文字

### 装飾ルール
- 装飾線は **左上のセクションラベル横の1本のみ**（24px の細線、`#0A0A0A`）
- 蛍光ペン強調（青または橙の太線）は **1スライドに1ヶ所のみ**、必ず文字の下端から **4px 以上下**に配置
- カードは角丸 16px、影なし、境界線は `#E8E4DA` 1px
- 数字に下線・装飾は**つけない**（数字の大きさだけで強調）
